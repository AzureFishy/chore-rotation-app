Quick Start

  1. Add family members in the Settings section
  2. Add chores that need to be done regularly
  3. Set your preferred start date
  4. Click "Assign New Chores" to distribute tasks
  5. Use "Today" to see current assignments

Tips & Tricks

  • The system ensures everyone does each chore once per cycle
  • A new cycle starts when all chores are completed by everyone
  • Use the date display toggle to show/hide due dates
  • Navigate to any week/cycle using the number inputs
  • Changing the seed creates new fair assignments
