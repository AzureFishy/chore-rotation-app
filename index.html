      Quick Start

      1. Add family members in the Settings section
      2. Add chores that need to be done regularly
      3. Click "Assign New Chores" to distribute tasks
      4. Use "Today" to jump to the current date
      5. Use "Reset" to start over from week 1
      6. Track completion using the green colored text

            Quick Start

            1. Add family members in the Settings section
            2. Add chores that need to be done regularly
            3. Click "Assign New Chores" to distribute tasks
            4. Use "Today" to jump to the current date
            5. Use "Reset" to start over from week 1
            6. Track completion using the green colored text

            Tips & Tricks

            • The system automatically tracks which chores each person has done in a cycle
            • When everyone has done all chores once, a new cycle begins!
            • Use Export/Import to save or transfer your settings
            • You can manually navigate to any week/cycle using the number inputs
            • Changing the seed will create different random assignments while maintaining fairness
            • The history section shows the last 7 chores for each person
            • Green text indicates current assignments
            • Strikethrough text shows completed chores from previous weeks
